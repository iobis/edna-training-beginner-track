Datos de ejemplo

El conjunto de datos con el que trabajaremos es un conjunto de datos de metabarcoding de eDNA, obtenido de eDNA Expeditions. Este ejemplo está adaptado de materiales de capacitación desarrollados para OBON en 2024, creados por Pieter Provoost, y se puede encontrar en https://github.com/iobis/obon-2024-dna-training.

Puede descargar los archivos del conjunto de datos directamente desde https://github.com/iobis/edna-training-beginner-track/tree/main/data, o usar el siguiente código para descargarlos:

# Replace with the actual data download for this lesson
download.file(
  url = "https://github.com/iobis/edna-training-beginner-track/tree/main/data",   # replace with actual URL
  destfile = here("data", "edna-data.csv")
)
NotaSobre el conjunto de datos de ejemplo

Hay seis archivos en la carpeta \data: cuatro archivos de salida del pipeline de ADN (samples.csv, seqtab.csv, sequences.fasta, y taxonomy.txt), un archivo de diccionario de datos que hemos creado para proporcionar definiciones de cada columna en el archivo samples, y un archivo methods.txt que contiene texto libre que suele incluir metadatos e información importante que registraremos en la tabla de la extensión DNA Derived Data.
